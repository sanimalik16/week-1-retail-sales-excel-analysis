# Project Title
Retail Sales Data Cleaning & Basic Analysis in Excel 

## Project Objective
The objective of this project is to understand, clean, analyze and visualize a retail sales dataset using Microsoft Excel. The task focused on preparing raw data for analysis.

## Tools Used
- Microsoft Excel
- GitHub

## Dataset Overview
The dataset includes the following columns: 
- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount 

## Task Completed

### Data Understanding
- Identified the number of rows and columns.
- Checked data types of **Date**, **Product Category** and **Total Amount**.
- Identified 3 unique product categories.
- Determined the minimum and maximum age of the customer.

### Formatting Data
- Created a new sheet named **cleaned_data** and copied raw data into it.
- Converted raw data into a table.
- Changed the data type of **Data**, **Price per Unit** and **Total Amount** column to Date and Currency respectively.
- Adjusted the width of columns.
- Formatted headers to improve readability.
- Froze the first row of the dataset.

### Data Quality Checks 
- Checked for null and duplicate values.
- Validated the values of **Quantity** and **Price per Unit** column.
- Compared the **Total Amount** to **Calculated Column**  using formulas.

### Basic Analysis
Use Excel formulas to calculate the following: 
1. Total sales amount 
2. Average sales amount 
3. Minimum sales amount 
4. Maximum sales amount 
5. Total quantity sold 
6. Average customer age 
7. Youngest customer age 
8. Oldest customer age 
9. Total number of transactions 
10. Total number of unique product categories

Created the following summary tables:
- Product Category Total Sales
- Product Category Total Quantity Sold
- Sales by Gender

### Sorting and Filtering 
Took screenshots of the following filters:
- Transactions where Product Category is Clothing.
- Transactions where Gender is Female.
- Transactions where Quantity is greater than 2.

### Charts
Created the following charts: 
- Bar chart: Total Sales by Product Category
- Column chart: Total Quantity Sold by Product Category
- Pie chart: Total Sales by Gender 

### Observations
1. Electronics generated the highest sales: $156,905.
2. Female customers generated slightly more sales than males.
3. Total quantity sold is 2514.
4. The highest transaction amount is $2000.
5. Average age of customers is approx. 41 years.

## Learning Outcomes
After completing this project, I gained experience in data cleaning, formatting data, using excel formulas, sorting and filtering, creating summary tables and building charts.
